## ⬇️ Download
![Downloads](https://img.shields.io/github/downloads/YonY18/Terminal_Downloader_Releases/total?style=for-the-badge&color=red)

1. Go to the **Releases** section:

   👉 https://github.com/YonY18/Terminal_Downloader_Releases

2. Download the latest version:

   ```
   Terminal_Downloader_vX.X.X.exe
   ```

3. Double-click to run.

> ⚠ Windows may show a SmartScreen warning because the app is not code-signed.  
> Click **More info → Run anyway**.

## 🔐 Security

Each release includes a SHA256 hash for integrity verification.

If you want to verify the file manually:

```powershell
Get-FileHash .\Terminal_Downloader_vX.X.X.exe -Algorithm SHA256
```

Compare it with the hash published in the release notes.
